Instalacion

Listo en 3 pasos. Compatible con Claude Desktop, Claude Code, Cursor y cualquier cliente MCP.

1

Clonar e instalar

Clona el repositorio e instala las dependencias.

# Clonar git clone https://github.com/spirit122/commercehub-mcp.git cd commercehub-mcp # Instalar y compilar npm install npm run build
2

Configurar credenciales

Copia el archivo de ejemplo y agrega tus API keys.

cp .env.example .env # Edita .env con tus credenciales SHOPIFY_STORE_URL=https://tu-tienda.myshopify.com SHOPIFY_ACCESS_TOKEN=shpat_xxxxx STRIPE_SECRET_KEY=sk_live_xxxxx
3

Conectar con Claude

Agrega a tu configuracion de Claude Desktop o Code.

// claude_desktop_config.json { "mcpServers": { "commercehub": { "command": "node", "args": ["dist/index.js"] } } }

Ejemplos de uso

Habla con tu agente de IA en lenguaje natural. CommerceHub ejecuta las operaciones por ti.

"Muestrame las ventas de hoy comparadas con ayer"
Usa analytics_revenue con period=today y compare_previous=true. Retorna ingresos, ordenes, ticket promedio y % de cambio.
"Que productos necesito reabastecer?"
Usa inventory_low_stock + inventory_forecast para detectar productos criticos y predecir cuando se agotan.
"Sincroniza mi catalogo de Shopify a WooCommerce"
Usa products_sync con dry_run=true primero. Muestra los cambios y luego aplica con dry_run=false.
"Procesa el reembolso de la orden #1234"
Usa orders_refund con el order_id. Soporta reembolso total, parcial, y restock automatico.
"Quienes son mis clientes VIP?"
Usa customers_segments para ver la segmentacion automatica: VIP, Regular, En Riesgo, Perdido, Nuevo.
"Dame el reporte ejecutivo del dia"
Usa el prompt daily-report que combina revenue, ordenes, top productos, alertas de stock y recomendaciones.
